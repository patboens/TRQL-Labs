process(): Processes the API call by invoking a handler function (handlerFnc)

Abstract:

An APIHandler is invoked by the SmallAPIGW object upon service call.

You start defining as many handlers as you need and state which services it is supposed to handle.

Then you create a SmallAPIGW object to which you pass all handlers it may invoke to handke the calls.

Then you ask the SmallAPIGW to process the call, which really means to dispatch the call to the proper APIHandler. The API Handler that has in charge the service is then called via its process() method.

It does its job and returns the answer to the SmallAPIGW which is then responsible to pack the response in a proper structure that will eventually be returned to the client.

API return codes are all maintained in trql.APIHandler.defines.php. Please refer to the list of HTTP status codes for an exhaustive list of possible values https://en.wikipedia.org/wiki/List_of_HTTP_status_codes.

Syntax:

process( $aCall )

Parameters:
Name Type Description
$aCall array Associative array that sketches the API call. Received by reference; possibly updated upon return

Return:

(array): API response (if any), or null if no handler defined for the service.

Example(s):


  use \trql\vaesoli\Vaesoli           as v;
  use \trql\apigateway\APIHandler     as APIHandler;
  use \trql\apigateway\SmallAPIGW     as SmallAPIGW;

  $szSelf = null;

  {   /* Setting $szSelf */
      if     ( isset( $_SERVER['PHP_SELF'] ) )
      {
          $szSelf = $_SERVER['PHP_SELF'];
      }
      elseif ( isset( $_SERVER['REQUEST_URI'] ) )
      {
          $szSelf = $_SERVER['REQUEST_URI'];
      }
      elseif ( isset( $_SERVER['SCRIPT_NAME'] ) )
      {
          $szSelf = $_SERVER['SCRIPT_NAME'];
      }
  }   /* Setting $szSelf */


  /* Specialized handler: Quitus/TRQL Labs */
  $oQuitusHandler             = new APIHandler( 'accounts,customers,customer,invoices,invoice,offers,offer' );
  $oQuitusHandler->handlerFnc = 'quitusHandlerFnc';           /* Function to execute */

  $oGW                        = new SmallAPIGW( array( $oQuitusHandler ) );
  $oGW->verbSlot              = 2;                            /* Slot in 'parts' where the verb (service name) is supposed to be */

  // The Gateway will handle it and dispatch it to the proper handler
  $aResponse                  = $oGW->processCall( $szSelf ); /* Process the call */

  // The Gateway will send the response to the client
  // Here, it will either send it in XML or in JSON
  // depending on the query string
  $oGW->send( $aResponse,array( 'type' => ( isset( $_GET['xml'] ) ? 'xml' : 'json' ) ) );

  function quitusHandlerFnc( &$aCall,&$oHandler )
  {
      $tStart     = microtime( true );
      $szVerb     = $aCall['verb'];

      // Prepare a default response
      $aResponse  = array( 'request'  => $aCall['parts']          ,
                           'verb'     => $szVerb                  ,
                           'matchAt'  => -1                       ,
                           'retcode'  => API_ERROR_FORBIDDEN      ,
                           'response' => null                     ,
                           'perf'     => -1                       ,
                         );

      switch ( strtolower( $szVerb ) )
      {
          // Fictitious API service
          case 'accounts' :
              {
                  // Start by saying at which line you actually matched the
                  // service request (ideal for debugging situations)
                  $aResponse['matchAt'] = __LINE__;

                  // Then overwrite the API_ERROR_FORBIDDEN error code
                  // by API_ERROR_BAD_REQUEST : you will now need to
                  // check the parameters to determine whether the call was
                  // well-formed.
                  $aResponse['retcode'] = API_ERROR_BAD_REQUEST;
                  
                  // HERE YOU CHECK THE PARAMETERS (see how $aCall) is formed

                  // If all OK, you send the response with a code similar to:

                  $aResponse['response'] = "All OK";

                  // If NOT OK, you send the response with a code similar to:

                  $aResponse['response'] = "Error description";
              }
              break;

              }

      }
