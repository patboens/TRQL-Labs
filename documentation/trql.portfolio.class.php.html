Class: Portfolio Description: A catalog of aspirations (projects) Declaration: class Portfolio extends Catalog Properties:
Name Type Description
$self array Fixed 'class' information.
$wikidataId string wikidataId. Container for documents. There is also another notion that comes close to what TRQL Labs calls a portfolio: unbound collection of visual artworks housed in a binder, folder or other container ('Q79509036')
$aAspirations array An array of Aspirations belonging to the Portfolio
$szStorageFolder string A relative path where all Aspirations get stored
$aStorageMap array Set of values that depicts where the portfolio gets stored.
$modulo int Prime number used to determine the exact storage location of the portfolio (see trql.storage.trait.php).
$aDomains array Array of domains pertaining to the Portfolio (e.g. "payments", "cards", ...)
$oMission Mission The Portfolio mission (if any).
$szCodeName string Code name of the Portfolio.
