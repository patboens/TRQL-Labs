Class: Question Description: Properties:
Name Type Description
$self array Fixed 'class' information.
$acceptedAnswer Answer|ItemList The answer(s) that has been accepted as best, typically on a Question/Answer site. Sites vary in their selection mechanisms, e.g. drawing on community opinion and/or the view of the Question author.
$answerCount Integer The number of answers this question has received.
$downvoteCount Integer The number of downvotes this question, answer or comment has received from the community.
$suggestedAnswer Answer|ItemList An answer (possibly one of several, possibly incorrect) to a Question, e.g. on a Question/Answer site.
$upvoteCount Integer The number of upvotes this question, answer or comment has received from the community.
