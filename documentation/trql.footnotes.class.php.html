Class: Footnotes Description: A Footnotes object gathers notes in an array. The full array of notes can then be rendered later at some point. Notes can be added in the flow of a text. Each note is replaced by a superscript reference that points to the actual note. When notes are all rendered, it is easy to go back to the original text where the note was added in the first place. Declaration: class Footnotes extends WebPageElement Properties:
Name Type Description
$self array Fixed 'class' information.
$aNotes array An array of notes that must be rendered at some point
$szPrefix string A prefix that will be added to the note counter when it is rendered
$iTeaserLength int The length of the teaser
$iStart int Counter start. 0 by default. DO NOT USE NEGATIVE NUMBERS.
